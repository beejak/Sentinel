# Security Policy

## Supported Versions
We support the latest commit on `main`. No long-term support branches at this time.

## Reporting a Vulnerability
- Please email security reports to security@example.invalid with details and a minimal reproduction if possible.
- Alternatively, open a private Security Advisory in GitHub (Security > Advisories) for this repo.
- We aim to acknowledge within 72 hours and provide an ETA for a fix after triage.

## Disclosure
- We prefer coordinated disclosure. We will credit reporters in release notes if desired.
